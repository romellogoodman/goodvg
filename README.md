# goodvg

A library for scripting html. [Learn more](https://mellogood.substack.com/p/goodgraphicsjs) about this library.

## Use

```js
const svg = new Graphic({
  attributes: {
    fill: "white",
    style: "background: #1b1b1b",
  },
});

svg.circle("50%", "50%", 50);

svg.draw();
```
